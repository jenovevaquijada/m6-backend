# 🛒 Mi Tiendita - E-commerce Full Stack
Este proyecto es una tienda virtual funcional que integra un frontend responsivo con un backend en Node.js, cumpliendo con los estándares de persistencia de datos y diseño empático.

Tagline: "Construyendo experiencias digitales desde la empatía y la creatividad."

## 🚀 Funcionalidades Principales
Catálogo Dinámico: Grid simétrico (4x2) con precios en CLP.

Carrito de Compras: Cálculo de subtotal, IVA (19%) y total en tiempo real.

Gestión de Stock: Descuento automático del inventario al vender.

Persistencia: Almacenamiento en archivos JSON (productos.json y ventas.json).

IDs Únicos: Uso de la librería uuid para transacciones.

## 🛠️ Tecnologías
Node.js, Express.js, File System (fs), HTML5, CSS3, JavaScript.

## 📦 Instrucciones 
Abrir la terminal en la carpeta del proyecto.

Ejecutar npm install para instalar dependencias.

Iniciar el servidor con node server.js.

Abrir en el navegador: http://localhost:3000.

Nota: La carpeta node_modules fue excluida por buenas prácticas de desarrollo.
